Internal CSS ნიშნავს სტილების განთავსებას **HTML დოკუმენტის შიგნით**, "head" თეგში, "style" ელემენტში. ეს მეთოდი გვაძლევს საშუალებას **ერთი ფაილით** გავაკონტროლოთ შიგთავსის ვიზუალური ელემენტები.

Inline CSS ნიშნავს **სტილების პირდაპირ დოკუმენტის ელემენტზე დამატებას**, მაგალითად, **"style" ატრიბუტით**. მისი გამოყენება უფრო **პირდაპირია**, მაგრამ არ არის საუკეთესო გადაწყვეტილება დიდი და კომპლექსური საიტებისთვის.

External CSS გულისხმობს **განსაკუთრებული ფაილის გამოყენებას** (მაგალითად, "styles.css"), სადაც ყველა სტილი **ცალკე** აღწერილია. ეს მეთოდი უფრო **პროფესიონალურია**, რადგან **მარტივად იყენებს მრავალ გვერდზე ერთდროულად**.
